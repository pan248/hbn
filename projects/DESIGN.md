# DESIGN.md

## 气质与意象
晨光透过亚麻纱帘洒在极简空间，空气中漂浮着微尘，一切都在缓慢流动。HBN「慢一点远一点」——护肤即修行，慢下来才能走更远。画面质感接近胶片摄影的温润颗粒感，呼吸之间有时间沉淀的温度。

## 视觉策略
- 摄影方向：低饱和暖调、自然光、微距肌肤纹理 + 远景辽阔地平线，形成"近/远"对照
- 图形语言：极细线条圆环（时间/循环意象）、微弧分割线、留白占大面积
- 避免硬边几何与强对比块面，以柔和渐变和有机形态为主

## 字体排版
- 中文标题：Noto Serif SC（Google Fonts .cn 域），衬线传递沉稳与仪式感
- 正文/辅助：系统默认无衬线
- 排版节奏：标题大字距宽松，正文适中行高（1.8），留白即呼吸

## 动效与交互
- 运动性格：缓慢、柔和、有呼吸感
- 缓动曲线：cubic-bezier(0.25, 0.1, 0.25, 1.0) — 近似 ease 但更柔
- 滚动触发：元素从 opacity 0 + translateY(30px) 缓缓浮入，持续 1.2s
- 数字/文字逐帧显现：模拟慢速书写感
- 浮动粒子：极淡金色圆点以 20-40s 周期缓慢飘动
- 禁用弹跳、抖动、快速闪烁

## 页面结构
1. **全屏 Hero** — 品牌名 + 主题语 + 氛围背景渐变 + 缓慢浮动光点
2. **哲学引言** — "慢"的诠释，大段留白，衬线排版
3. **远近对照** — 左/上「近」肌肤微距 vs 右/下「远」辽阔远景，视觉叙事
4. **品牌故事** — HBN 成分哲学，竖向时间线布局
5. **尾屏 CTA** — 二维码 + 分享引导 + 品牌标志

## 设计禁忌
- 禁止荧光色、高饱和蓝紫渐变、科技感光效
- 禁止圆角卡片堆叠的"模板感"布局
- 禁止大段黑色硬底 + 白色文字的强对比块
- 禁止快速弹入弹出动画
- 禁止使用 emoji
